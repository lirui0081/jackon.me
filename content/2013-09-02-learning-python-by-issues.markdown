#### 任务：统计一句话中各英文单词的出现频率

text 变量中存储的是一句话，
这句话中可能包含多个子句，子句之间用 ',' 分割开。
单词与单词之间用空格隔开。
比如：
"here is a string, this is a text, this is a sentence"

请统计这句话中各英文单词出现的频率，将结果存储为字典。
字典的 key 是单词，value 是出现频率。
